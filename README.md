# 로생상담소

Lost Ark 캐릭터 정보를 조회하고 장비, 아크 패시브, 스킬, 보석, 아바타, 악세/각인 효율을 분석하기 위한 Next.js 프로젝트다.

## 문서

- [개발 일지](./docs/development-log.md)
- [로스트아크 데미지 계산 기준](./docs/lostark-damage-formula.md)
- [다음 작업 메모](./NEXT_TASKS.md)

## 개발 명령

```bash
npm run dev
npm run dev:restart
npm test
npm run lint
npm run build
```

## 환경 변수

`.env.example`을 참고해 `.env.local`을 만든다. `.env.local`은 Git에 올리지 않는다.
