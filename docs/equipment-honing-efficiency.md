# 장비 강화 효율 기준

## 범위

- 기준 무기: 세르카 고대 무기.
- 기준 방어구: 세르카 고대 방어구.
- 시작 강화: 11강.
- 최대 강화: 25강.
- 11강 무기 공격력: 167,706.
- 무기 강화는 무기 공격력 증가가 기본 공격력/전투력에 주는 순수 증가율까지 정리한다.
- 방어구 강화는 주스탯 증가와 LoaCalc 기준 재료/확률 데이터를 정리한다.
- 강화 성공률, 장기백, 재료 수량, 숨결 투입 효율은 LoaCalc 기준값을 사용한다.

## 계산식

현재 전투력 모델에서 주스탯과 다른 배율이 고정이면, 무기 공격력 증가는 기본 공격력에 제곱근으로 반영된다.

```text
강화 1회 강해짐 = sqrt(다음 무기 공격력 / 현재 무기 공격력) - 1
11강 대비 누적 강해짐 = sqrt(해당 강화 무기 공격력 / 11강 무기 공격력) - 1
```

주의:

- 이 값은 무기 공격력만 바뀐다고 보는 기준값이다.
- 주스탯, 무기 공격력 %, 공격력 %, 추가 피해, 보석, 각인, 아크패시브, 팔찌 특수옵션은 고정으로 본다.
- 방어구 강화는 방어력도 올리지만 전투력 계산에는 주스탯 증가만 반영한다.
- 실제 캐릭터 총 전투력 증가는 다른 배율과 비용 모델을 결합한 뒤 계산한다.

## 세르카 고대 방어구 11강 주스탯 기준

방어구는 부위마다 11강 기준 주스탯이 다르다. 아래 값은 투구, 어깨, 상의, 하의, 장갑 순서의 기준값이다.

| 부위 | 11강 주스탯 |
|---|---:|
| 투구 | 96,801 |
| 어깨 | 103,023 |
| 상의 | 77,441 |
| 하의 | 83,664 |
| 장갑 | 116,161 |
| 합계 | 477,090 |

방어구 강화 효율은 주스탯 증가분을 아래 식으로 계산한다.

```text
방어구 강화 1회 강해짐 = sqrt(강화 후 총 주스탯 / 강화 전 총 주스탯) - 1
```

주의:

- 부위 단독 주스탯 비율이 아니라 캐릭터의 총 주스탯 기준으로 계산한다.
- 강화 후 총 주스탯은 기존 총 주스탯에서 해당 부위의 강화 증가분만 더한 값이다.
- 상의, 하의, 장갑 증가분은 예측값이므로 실측값이 들어오면 교체한다.

## 다음 계산에 필요한 값

강화 비용 계산에는 단계별로 아래 값을 사용한다.

- 기본 성공률.
- 장인의 기운 증가량과 장기백 도달 규칙.
- 1회 시도 재료 수량: 운명의 파괴석 결정/운명의 수호석 결정, 위대한 운명의 돌파석, 상급 아비도스 융화제, 운명의 파편, 골드.
- 추가 재료 수량과 확률 보정: 용암의 숨결/빙하의 숨결.
- 운명의 파편 주머니를 파편 개수 기준 단가로 환산하는 규칙.

각 단계별로 `기대 비용 / 강해짐 %`와 `숨결 투입 시 기대 비용 / 강해짐 %`를 비교한다.

### 확률 진행 규칙

LoaCalc `refine.ts`에서 우리 계산에 필요한 규칙만 정리한다.

```text
JANGIN_ACCUMULATE_DIVIDER = 2.15

트라이 성공률 = round4(min(현재 기본 확률 + 추가 확률 + 숨결 확률, 100%))
실패 후 기본 확률 = min(현재 기본 확률 + 최초 기본 확률 * 0.1, 최초 기본 확률 * 2)
실패 후 장인의 기운 증가 = 트라이 성공률 / 2.15 * 장인의 기운 배율
장인의 기운 >= 100%면 다음 트라이는 확정 성공
```

세르카 고대 무기 `weapon.t4_1730`과 방어구 `armor.t4_1730`에는 원본 기준 별도 성장지원/영지연구/골드감소 규칙이 붙지 않는다.

따라서 기본값:

| 항목 | 값 |
|---|---:|
| 추가 확률 | 0% |
| 장인의 기운 배율 | 1 |
| 재료 수량 감소 | 0% |
| 골드 감소 | 0% |

### 비용 계산 입력 규칙

1회 시도 비용:

```text
기본 시도 비용 =
  기본 재료 수량 * 단가
  + 위대한 운명의 돌파석 수량 * 단가
  + 상급 아비도스 융화제 수량 * 단가
  + 운명의 파편 수량 * 단가
  + 골드

숨결 투입 비용 =
  숨결 사용량 * 숨결 단가

트라이 비용 = 기본 시도 비용 + 숨결 투입 비용
```

기본 재료와 숨결 종류:

| 장비 | 기본 재료 | 숨결 |
|---|---|---|
| 무기 | 운명의 파괴석 결정 | 용암의 숨결 |
| 방어구 | 운명의 수호석 결정 | 빙하의 숨결 |

기대 비용은 성공 시 종료, 실패 시 다음 실패 누적 상태로 재귀 계산한다.

```text
기대 비용 = 현재 트라이 비용 + (1 - 트라이 성공률) * 실패 이후 기대 비용
```

숨결 사용량은 `0개`부터 `최대 개수`까지 비교한다. 최종 구현에서는 각 단계별로 노숨, 풀숨, 최적 숨결 수를 모두 계산하고 `비용 / 강해짐 %`를 비교한다.
