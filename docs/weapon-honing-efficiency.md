# 무기 강화 효율 기준

## 범위

- 기준 무기: 세르카 고대 무기.
- 시작 강화: 11강.
- 최대 강화: 25강.
- 11강 무기 공격력: 167,706.
- 이 문서는 무기 공격력 증가가 기본 공격력/전투력에 주는 순수 증가율만 정리한다.
- 강화 성공률, 장기백, 재료 수량, 숨결 투입 효율은 별도 단계에서 붙인다.

## 계산식

현재 전투력 모델에서 주스탯과 다른 배율이 고정이면, 무기 공격력 증가는 기본 공격력에 제곱근으로 반영된다.

```text
강화 1회 강해짐 = sqrt(다음 무기 공격력 / 현재 무기 공격력) - 1
11강 대비 누적 강해짐 = sqrt(해당 강화 무기 공격력 / 11강 무기 공격력) - 1
```

주의:

- 이 값은 무기 공격력만 바뀐다고 보는 기준값이다.
- 주스탯, 무기 공격력 %, 공격력 %, 추가 피해, 보석, 각인, 아크패시브, 팔찌 특수옵션은 고정으로 본다.
- 실제 캐릭터 총 전투력 증가는 다른 배율과 비용 모델을 결합한 뒤 계산한다.

## 다음 계산에 필요한 값

무기 강화 비용 계산에는 단계별로 아래 값이 필요하다.

- 기본 성공률.
- 장인의 기운 증가량과 장기백 도달 규칙.
- 1회 시도 재료 수량: 운명의 파괴석 결정, 위대한 운명의 돌파석, 상급 아비도스 융화제, 운명의 파편, 골드.
- 추가 재료 수량과 확률 보정: 용암의 숨결.
- 운명의 파편 주머니를 파편 개수 기준 단가로 환산하는 규칙.

이 값들이 들어오면 각 단계별로 `기대 비용 / 강해짐 %`와 `숨결 투입 시 기대 비용 / 강해짐 %`를 비교한다.
