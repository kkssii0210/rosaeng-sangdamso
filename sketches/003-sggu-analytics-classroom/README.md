## Variant: Sggu Analytics Classroom

### Design stance
Clean Analytics Dashboard의 전문적인 정보 구조와 Sggu Classroom Green의 캐릭터 설명력을 결합한 하이브리드 시안입니다.

### Key choices
- Layout: 좌측 분석 내비게이션 + 중앙 칠판 브리핑/추천 대시보드 + 우측 슥구 상담 패널
- Tone: 백엔드 계산 결과는 대시보드처럼 조밀하게 보여주고, 추천 이유는 칠판 판서처럼 쉽게 설명
- Color: 따뜻한 중립 배경, 녹색 칠판, 절제된 블루/앰버 포인트를 함께 사용
- Interaction: 조회, 탭, 리포트 복사, 상담 요청 버튼의 가벼운 toast 피드백

### Best for
프로젝트의 메인 결과 화면 또는 전투력 효율 상담 화면을 개편할 때 기준 시안으로 사용하기 좋습니다.
