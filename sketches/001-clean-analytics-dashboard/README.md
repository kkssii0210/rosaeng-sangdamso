## Variant: Clean Analytics Dashboard

### Design stance
로생상담소를 “따뜻하지만 전문적인 분석 도구”로 보이게 하는 대시보드형 시안입니다.

### Key choices
- Layout: 좌측 분석 메뉴, 중앙 결과 대시보드, 우측 슥구 상담 패널의 3-column 구조
- Typography: 큰 숫자와 짧은 라벨 중심으로 스캔 가능성 우선
- Color: DESIGN.md의 중립/블루 계열을 유지하고, 따뜻함은 약한 앰버/그린 상태 색으로만 보조
- Interaction: 메뉴/탭 active 전환, 조회/공유/상담 버튼 toast 피드백

### Trade-offs
- Strong at: 포트폴리오에서 백엔드 분석 결과와 AI 상담 경계를 설명하기 좋음
- Weak at: 감성적인 상담소 분위기는 캐릭터 중심형보다 약함

### Best for
- 캐릭터 분석 결과, 스펙업 효율, LLM 상담 근거를 한 화면에서 보여주는 메인 결과 화면
