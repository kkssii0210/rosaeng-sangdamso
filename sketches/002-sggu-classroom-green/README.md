## Variant: Sggu Classroom Green

### Design stance
슥구가 칠판 앞에서 스펙업 효율을 강의해주는 “친근한 강의실” 콘셉트입니다.

### Key choices
- Layout: 중앙 칠판/강의 무대 + 우측 수업 보조 패널
- Typography: 칠판 위 큰 강의 제목과 판서형 공식으로 설명 중심 구성
- Color: 녹색 칠판, 연두빛 배경, 나무 프레임으로 강의실 분위기 강화
- Interaction: 자료 보기, 예제 다시 듣기, 숙제 버튼 클릭 시 toast 피드백

### Trade-offs
- Strong at: 슥구 캐릭터성과 설명형 AI 상담의 차별점이 잘 드러남
- Weak at: 순수 분석 대시보드보다 정보 밀도와 전문 도구 느낌은 낮음

### Best for
- 첫 방문자 온보딩, 분석 결과 설명 화면, “왜 이 추천인지”를 쉽게 풀어주는 튜토리얼 UI
